Usando links externos

Podemos usar links de outros sites, por isso o nome é "externo". Para isso, utilizamos a tag <a> que significa Âncora. Também devemos fazer com que o link se abra numa nova aba, para o usuário não deixar de voltar ao seu site. Para isso, usamos os seguintes comandos: <target"_blank" rel="external"> Passos:

  1. usar a tag <a>
  2. colocar o link na <href>
    3. referência a hipertexto
  3. dentro da tag <a> adicionar <target"_blank" rel="external">

Exemplos:

  1. (Sem o comando <target"_blank" rel="external">)
    Para acessar o canal do youtube do professor Guanabara, clique neste link e encontre diversos cursos na área de tecnologia.

  2. (com o comando <target_"blank" rel="external">)
    Para acessar ao repositório do Github do professor Gustavo Guanabara, clique aqui e encontre seus materiais de estudos.
