Negrito e Itálico do jeito certo

Negrito

Para colocar um termo em negrito usa-se a TAG <strong>

Itálico/Ênfase

Para usar um termo em Itálico usa-se a tag <EM >

Texto Marcado

Podemos também, criar um mark usando a tag <mark>

Ou então, marcar todos de uma vez criando um parágrafo que comandará todas as marcações, esse parágrafo fica embaixo de títle e é feito da seguinte maneira: <style, mark{ background-color }>

Texto deletado

Podemos deletar um texto deletado usando a tag <del>

Textos grandes

De forma obsoleta, utilizamos a tag <big>

Textos pequenos

De forma semântica, utilizamos a tag <small> para diminuir o texto

Texto inserido ou "sublinhado"

⇾ Usando a tag obsoleta <u>

⇾ Usando a tag semântica e adequada <ins>

Texto sobrescrito

Para mark coisas do tipo x20+3 utilizamos a tag <sup>

texto subscrito

Para escrever coisas como H2O utilizamos a tag <sub>
